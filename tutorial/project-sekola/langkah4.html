# Langkah keempat - Membuat table admin, guru, siswa, kelas dan distKelas

Perhatian

Pastikan sudah login ke mysql seperti yang sudah dijelaskan pada Langkah ketigaopen in new window.

Cek apakah database sekola sudah dibuat atau belum dengan menuliskan perintah SHOW DATABASES;

Sebelum menambahkan table, kita harus masuk terlebih dahulu ke dalam databasenya, dengan perintah USE nama_database;, dalam kasus ini, maka kita ketikkan perintah

USE sekola;

# Format syntax membuat table baru dalam database

Berikut adalah format syntax SQL yang digunakan untuk membuat table baru di dalam database:

CREATE TABLE nama_table(
    column data_type constraint,
    column data_type constraint,
    column data_type constraint,
    . . . .
    column_ke-n data_type_ke-n constraint,
    PRIMARY KEY(column)
);

Keterangan:

nama_tablemerupakan nama table yang ingin dibuat
columnmerupakan nama field/column dalam table
data_typemerupakan tipe data dari field/column
PRIMARY KEY(column)menentukan field/column mana yang akan dijadikan PRIMARY KEY
constraintuntuk mengatur rule pada data atau table, contoh mengatur agar field boleh kosong atau tidak boleh kosong

# Membuat table admin

Sesuai dengan gambar desain database pada Langkah ketigaopen in new window, table admin memiliki deskripsi sebagai berikut:

Table: admin

NoFieldData TypeKeterangan
1id_adminINT(10)Primary Key dari table admin, Tidak boleh kosong
2nama_adminVARCHAR(100)Tidak boleh kosong
3jk_adminINT(1)Jenis kelamin, diwakili oleh angka 0 untuk perempuan atau 1 untuk laki-laki, Tidak boleh kosong
4agama_adminVARCHAR(10)Tidak boleh kosong
5alamat_adminTEXTTidak boleh kosong
6noTelp_adminVARCHAR(15)Tidak boleh kosong
7email_adminVARCHAR(50)Tidak boleh kosong
8username_adminVARCHAR(10)Tidak boleh kosong
9status_adminVARCHAR(10)Sebagai penanda bahwa admin masih aktif atau tidak aktif, Tidak boleh kosong
10admin_dibuatOlehVARCHAR(10)agar terlacak siapa yg membuat data admin baru, Tidak boleh kosong
11tglDibuat_adminDATETIMEagar terlacak kapan data admin baru dibuat, Tidak boleh kosong

Syntax untuk membuat table admin adalah

CREATE TABLE admin(
    id_admin INT(10) NOT NULL,
    nama_admin VARCHAR(100) NOT NULL,
    jk_admin INT(1) NOT NULL,
    agama_admin VARCHAR(10) NOT NULL,
    alamat_admin TEXT NOT NULL,
    noTelp_admin VARCHAR(15) NOT NULL,
    email_admin VARCHAR(50) NOT NULL,
    username_admin VARCHAR(10) NOT NULL,
    status_admin VARCHAR(10) NOT NULL,
    admin_dibuatOleh VARCHAR(10) NOT NULL,
    tglDibuat_admin DATETIME NOT NULL,
    PRIMARY KEY (id_admin)
);
