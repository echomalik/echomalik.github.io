Perhatian

# Membuat table materi

Sesuai dengan gambar desain database pada Langkah ketigaopen in new window, table materi memiliki deskripsi sebagai berikut:

Table: materi

NoFieldData TypeKeterangan
1id_materiINT(10)Primary Key dari table materi, Tidak boleh kosong
2judul_materiTEXTTidak boleh kosong
3deskripsi_materiTEXTTidak boleh kosong
4lampiran_materiTEXTBoleh kosong
5idKelasINT(10)Foreign Key dari tabel kelas, Tidak boleh kosong
6status_materiVARCHAR(10)Sebagai penanda bahwa materi masih aktif atau tidak aktif, Tidak boleh kosong
7materi_dibuatOlehVARCHAR(10)agar terlacak siapa yg membuat data materi baru, Tidak boleh kosong
8tglDibuat_materiDATETIMEagar terlacak kapan data materi baru dibuat, Tidak boleh kosong

Syntax untuk membuat table materi adalah

CREATE TABLE materi(
    id_materi INT(10) NOT NULL,
    judul_materi TEXT NOT NULL,
    deskripsi_materi TEXT NOT NULL,
    lampiran_materi TEXT,
    idKelas_materi INT(10) NOT NULL,
    status_materi VARCHAR(10) NOT NULL,
    materi_dibuatOleh VARCHAR(10) NOT NULL,
    tglDibuat_materi DATETIME NOT NULL,
    PRIMARY KEY (id)
);
