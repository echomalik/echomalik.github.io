HTML - Form

Gambar di atas merupakan sebuah formulir log in yang digunakan untuk masuk ke sebuah sistem atau aplikasi.

Formulir merupakan sebuah elemen yang digunakan untuk menerima nilai masukan untuk berikutnya diolah oleh server.

Struktur Dasar

Tag utama yang harus ada pada saat membuat formulir dengan HTML adalah sebagai berikut

<form method="" action="">
    
</form>

Atribut method digunakan untuk mendeskripsikan metode pengiriman data ke server. Sedangkan atribut acrion digunakan untuk mendeskripsikan lokasi tujuan pengiriman data dari seluruh nilai masukkan pada formulir

Atribut method dan action belum dijelaskan pada materi ini.

Fieldset

Fieldset merupakan istilah untuk sebuah tempat penampungan pengelompokan inputan yang sejenis atau berkaitan.

Ketika jumlah inputan banyak, maka perlu adanya pengelompokkan inputan agar terlihat lebi rapi dan mudah dibaca oleh pengguna.

Berikut adalah contoh script yang penggunaan fieldset

<form action="" method="">
    <fieldset>
        <legend>Biodata Diri</legend>
    </fieldset>
</form>

Script tersebut akan menghasilkan tampilan sebagai berikut

Tampilan di atas hanyalah sebuah penampung saja, tidak bisa memasukkan nilai apapun ke dalamnya.

Tag legend digunakan untuk mendeskripsikan judul dari fieldsetnya dan hanya bisa digunakan di dalam fieldset saja

Input

Dalam HTML terdapat beberapa tipe inputan yang bisa digunakan sebagai tempat menerima nilai masukan, di antaranya

Tipe Deskripsi
text Untuk 1 baris teks sederhana
number Untuk angka
email Untuk email dengan format yang sesuai
password Untuk password dengan karakter yang disembunyikan
range Untuk nilai rentang
date Untuk tanggal (tahun-bulan-tanggal)
time Untuk waktu (jam:menit:detik)
color Untuk warna, Color Picker
submit Tampilan sebagai tombol, untuk mengirimkan semua data yang ada pada formulir
reset Tampilan sebagai tombol, untuk menghapus semua nilai yang ada pada formulir

Tipe yang lebih lengkap ada pada dokumentasi berikut https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input

Misalnya kita ingin membuat sebuah formulir sebagai berikut

Maka script yang akan kita susun adalah sebagai berikut
















<form action="" method="">
    <fieldset>
        <legend>Biodata Diri</legend>
        <br><br>
        <input type="text" name="namadepan" placeholder="Masukkan nama depan">
        <br><br>
        <input type="text" name="namabelakang" placeholder="Masukkan nama belakang">
        <br><br>
        <input type="text" name="tempatlahir" placeholder="Kota lahir">
        <br><br>
        <input type="date" name="tanggallahir">
    </fieldset>
    <br>
    <input type="submit" name="submit" value="Daftar">
</form>

Atribut name digunakan untuk mendeskripsikan nama penampung dari nilai yang dimasukkan. Nama tersebutlah yang akan dipanggil oleh server sebagai variabel.

Atribut placeholder merupakan kalimat keterangan yang dimunculkan di dalam field inputan.

Atribut value pada input bertipe submit merupakan nilai dari input tersebut sekaligus sebagai tulisan dari tombolnya.

Label

Setiap inputan idealnya memiliki judul atau nama yang menjelaskan tentang nilai apa yang harus dimasukkan ke dalamnya. Tidak semua inputan bisa menggunakan placeholder.

Terdapat tag yang bisa digunakan untuk membuat sebuah label nama inputan dalam formulir, yaitu <label></label>

Terdapat beberapa cara untuk menerapkan label bersama dengan inputan yang ada

<!-- input ada dalam tag label -->
<label for="namadepan">
    Nama Depan
    <br>
    <input type="text" name="namadepan" id="namadepan">
</label>

<!-- label berdampingan dengan tag input-->
<label for="">Nama Depan</label>
<br>
<input type="text" name="namadepan" id="namadepan">

Nilai atribut for pada tag <label></label> harus sesuai dengan nilai id pada inputan pasangannya. Ini bertujuan agar ketika teks label diklik, maka text cursor akan langsung fokus ke inputan tersebut, septrti gambar berikut

Misalnya, kita ingin membuat form seperti berikut

Maka script yang akan kita buat adalah sebagai berikut

        <br>
        <label for="namadepan">
            Nama Depan
            <br>

        </label>
        <br><br>
        <label for="namabelakang">
            Nama Belakang
            <br>
            <input type="text" id="namabelakang" name="namabelakang" placeholder="Masukkan nama belakang">
        </label>        
        <br><br>
        <label for="tempatlahir">
            Tempat Lahir
            <br>
            <input type="text" id="tempatlahir" name="tempatlahir" placeholder="Kota lahir">
        </label>        
        <br><br>
        <label for="tanggallahir">
            Tanggal Lahir
            <br>
            <input type="date" id="tanggallahir" name="tanggallahir">
        </label>        
    </fieldset>
    <br>
    <input type="submit" name="submit" value="Daftar">
</form>

Radio

Input bertipe radio digunakan untuk memberikan sebuah pilihan yang hanya bisa dipilih salah satunya saja. Contohnya adalah jenis kelamin, antara laki - laki dan perempuan, maka pengguna hanya bisa memilih salah satunya saja. Berikut adalah contohnya

<label for="">Jenis Kelamin</label>
<br>
<label for="lakilaki">
    <input type="radio" id="lakilaki" name="jeniskelamin">Laki-laki
</label>
<br>
<label for="perempuan">
    <input type="radio" id="perempuan" name="jeniskelamin">Perempuan
</label>

Dengan script di atas, maka akan menghasilkan hasil sebagai berikut

Setiap opsi pilihan yang ada, harus memiliki nilai name yang sama, karena opsi - opsi tersebut berada pada satu kategori yang sama.

Checkbox

Checkbox merupakan kumpulan pilihan yang bisa dipilih lebih dari satu, contohnya seperti memilih hobi.

Contohnya adalah sebagai berikut

<label for="">Hobi</label>
<br>
<label for="membaca">
    <input type="checkbox" id="membaca" name="hobi">Membaca
</label>
<br>
<label for="menulis">
    <input type="checkbox" id="menulis" name="hobi">Menulis
</label>
<br>
<label for="maingame">
    <input type="checkbox" id="maingame" name="hobi">Main Game
</label>
<br>
<label for="fotografi">
    <input type="checkbox" id="fotografi" name="hobi">Fotografi
</label>

Sama seperti radio, nilai name pada checkbox harus sama jika opsi tersebut ada pada kategori yang sama.

Textarea

Jika input bertipe text hanya bisa diisi dengan teks satu baris, <textarea></textarea> dapat di isi dengan nilai lebih dari satu baris. Berikut adalah contohnya

<label for="alamat">
    Alamat Lengkap
    <br>
    <textarea id="alamat" name="alamat" cols="30" rows="10" placeholder="Masukkan alamat lengkap"></textarea>
</label>

Script tersebut akan menghasilkan tampilan sebagai berikut

Atribut cols dan rows digunakan untuk mendeskripsikan ukuran lebar dan tinggi textarea yang akan tampil di dalam formulir

Dropdown Selection

Menampilkan pilihan bisa juga menggunakan dropdown. Penggunaan dropdown bertujuan agar tampilan pilihan terlihat lebih ringkas.

Tag yang akan kita gunakan adalah <select></select>, dengan struktur dasar sebagai berikut

<select name="">
    <option value="opsi1">Opsi 1</option>
    <option value="opsi2">Opsi 2</option>
    <option value="opsi3">Opsi 3</option>
</select>

Reset

Selain tombol submit atau tombol kirim data, formulir pada HTML juga memiliki tombol reset yang digunakan untuk mengembalikan semua inputan dalam keadaan kosong kembali. Berikut contohnya

Maka tombol reset bisa kita tambahkan disamping tombol submit

<form action="" method="">
    <fieldset>
        <legend>Biodata Diri</legend>
        <input type="text" name="namadepan" placeholder="Masukkan nama depan">

File Form

Selain nilai dalam bentuk teks, formulir dalam HTML juga bisa menerima file sebagai nilai masukkan yang akan dikirim ke server.

Namun syaratnya adalah sebagai berikut

<form method="post" action="" enctype="multipart/form-data">
    . . .
</form>
  1. Metode pengiriman harus dalam bentuk post
  2. Menambahkan atribut baru ecntype dengan nilai multipart/form-data. Ini bertujuan agar formulir HTML bisa menerima inputan dalam bentuk file.

Misalnya kita ingin membuat formulir sebagai berikut

Maka script yang akan kita susun adalah sebagai berikut

<form method="post" action="" enctype="multipart/form-data">
    <fieldset>
        <legend>Berkas Lamaran</legend>
        <label for="cv">
            Unggah CV Terbaru
            <br>
            <input type="file" id="cv" name="cv">
        </label>
    </fieldset>
    <br>
    <input type="submit" name="submit" value="Unggah">
</form>

Dengan menekan tombol Choose File atau Browse, browser kita akan membuka jendela file manager untuk memilih file yang akan diupload.

Untuk membatasi pilihan dengan jenis file yang diiginkan, kita bisa menambahkan atribut accept dan mengisinya dengan nilai mime type yang diinginkan.

Misalnya, kita hanya akan menerima file dalam formt .pdf maka berikut adalah script yang kita akan buat

<input type="file" id="cv" name="cv" accept="application/pdf">

Dengan deskripsi mime type yang spesifik, file manager hanya akan menampilkan file dengan tipe tersebut.

Untuk melihat daftar seluruh mime type yang ada bisa dibaca dari dokumentasi berikut https://developer.mozilla.org/en-US/docs/Web/HTTP/Basics_of_HTTP/MIME_types/Common_types
